# WellNest — Smart Health & Fitness Companion

A full-stack web app built with React + Node.js/Express.

## Project Structure
```
wellnest/
├── backend/        ← Node.js + Express API
└── frontend/       ← React app
```

## Quick Start

### Step 1 — Backend
```bash
cd backend
npm install
node server.js
```
Server runs on http://localhost:5000

### Step 2 — Frontend (new terminal)
```bash
cd frontend
npm install
npm start
```
App opens at http://localhost:3000

## Features (All 5 Milestones)
- ✅ JWT Authentication (User & Trainer roles)
- ✅ Workout, Meal, Water & Sleep Trackers
- ✅ BMI Calculator + Daily Health Tips
- ✅ Analytics Dashboard with Charts
- ✅ Health Blog (create, like, comment)
- ✅ Trainer Matching System

## No database needed
All data is stored in-memory. Perfect for demo/prototype.
